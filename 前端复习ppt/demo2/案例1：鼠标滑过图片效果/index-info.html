制作背景渐变和图片移动动画

我们所看见的动画有的应用css3就可以完成，只要在css3中可以创建动画并引入动画。本例只是对css动画的一个简单应用。

效果源码

技术支持： 明日科技

源码

HTML

<div class="mr-bakg"> <div class="mr-picbom"> <div class="mr-pic"><img src="images/phine1.png" alt="" /></div> <div class="mr-picleft"><img src="images/phine2.png" alt="" /></div> <div class="mr-picright"><img src="images/phone3.png" alt="" /></div> </div> </div>

CSS

.mr-bakg{ margin:0 25%; width:800px; height:400px; border:1px #f00 solid; background:url(../images/01.jpg); animation:cs1 15s linear normal infinite; } @keyframes cs1{ 25%{ background:url(../images/01.jpg)} 50%{ background:url(../images/02.jpg)} 75%{ background:url(../images/03.jpg)} 100%{ background:url(../images/01.jpg)} } .mr-picbom{ position:relative; margin:0px 320px; padding-top:120px; width:220px; height:220px; } .mr-pic{ z-index:50;} .picleft,.mr-picright{ position: absolute; top: 120px; left: 0; } .mr-picbom:hover .mr-picleft{ transform: translateX(240px); display:block; transition:all 1s ease; } .mr-picbom:hover .mr-picright{ transform: translateX(-240px); display:block; transition:all 1s ease; }
